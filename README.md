# Confinject

Config inject tools, please read the source code before using

# As an user

Use [pipx](https://pypa.github.io/pipx/)

```
pipx install git+https://github.com/dinhanhx/confinject.git
```

# As a developer

with good old pip
```
pip install -e .
```

with fancy [poetry](https://python-poetry.org/)
```
poetry install
```
